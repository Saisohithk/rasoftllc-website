# RA SOFT LLC Website

## Stack
- **Frontend**: Angular 17 (standalone components) + Tailwind CSS
- **Backend**: Node.js + Express

## Development

### Backend
```bash
cd backend
npm install
npm run dev   # runs on http://localhost:3001
```

### Frontend
```bash
cd frontend
npm install
npm start     # runs on http://localhost:4200
```

## Build
```bash
cd frontend && npm run build
```
